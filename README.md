# Mi portafolio web

_Critical Coding for a Better Living._

**Descripción.**

Este proyecto es mi portafolio web personal donde muestro mis proyectos y habilidades como ilustradora. Incluye diferentes trabajos como bocetos, ilustraciones personales, organizados para presentar mi estilo artístico. El objetivo es crear un sitio accesible, responsive y fácil de navegar. 

## Purpose 

- **Mi objetivo:** el objetivo de este sitio es presentar mi trabajo como ilustradora y crear una presencia online donde pueda mostrar mis proyectos artísticos.

## Core Technologies (Deep Explanation)

### GitHub Pages

- Se usa para publicar el sitio web en internet.

### Git y GitHub

- Se utilizan para gestionar el código y guardar versiones del proyecto.

### JavaScript

- Se utiliza para gestionar pequeñas interacciones en la página, como el funcionamiento del menú de navegación.

## CSS

- Se encarga del diseño visual, incluyendo colores, tipografía, distribución y estilos para que el sitio sea atractivo.

## HTML5

Se utiliza para estructurar el contenido de la página web, como las secciones, imágenes y textos del portafolio.

## Diseño responsive

El sitio está adaptado para visualizarse en distintos dispositivos.

## Optimización de imágenes

Las ilustraciones se muestran en formato digital para su correcta visualización en la web.

Resolución de dudas técnicas, generar ideas, mejorar el proyecto.
